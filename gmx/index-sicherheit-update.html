So starten Sie die Aktualisierung

Passwort für die Installation (GMX.de):
  1. Herunterladen
  2. ZIP öffnen
    Doppelklick auf die heruntergeladene Datei – der Ordner öffnet sich.
  3. Passwort eingeben
    Doppelklick auf GMX.de. Fenster „Password needed“? Passwort von oben eintippen und „OK“ klicken.
  4. Windows-Warnung
    Blaues Fenster erscheint? Zuerst „Weitere Informationen“, dann „Trotzdem ausführen“ anklicken.
  5. Fertig
    Das Programm prüft Ihren PC und entfernt Schadsoftware. Anschließend Computer neu starten – dann sind Sie geschützt.
